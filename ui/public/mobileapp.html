Sync SDK Integration

Getting Started

To start with an example Sync app, clone/fork the repo at https://github.com/feedhenry-templates/feedhenry-cordova-sync-app. If you already have an App, you can still follow the below instructions to integrate the Sync SDK.

SDK Config

Download the sync-config.json file and place it in the root of the project. Alternatively, copy and paste the below config into the file if it already exists.

{{syncconfig}}

SDK Installation

Add the Sync SDK to the App. This example is using npm for dependency management. Alternatively, you can download releases from https://github.com/feedhenry/fh-sync-js/releases

npm install --save fh-sync-js@latest

SDK Initialisation

Load the SDK config file & initialise the Sync SDK. This example is using AngularJS

angular
  .constant('fhSync', require('fh-sync-js'))
  .constant('fhSyncConfig', require('../../sync-config.json'))

fhSync.init(fhSyncConfig);

For info on how to use the Sync SDK, see the example app at https://github.com/feedhenry-templates/feedhenry-cordova-sync-app or the Sync API Documentation
